## Overview

Font Awesome Alfred Workflow is built for Alfred on macOS and focuses on making Font Awesome icon lookup fast from the keyboard. It streamlines icon searching, cheatsheet reference access, and browsing across icon collections so you can stay inside Alfred instead of switching context.

The workflow is aimed at designers, developers, and anyone who works with icon sets and wants a quicker way to identify names, inspect references, and move through icon libraries. With Alfred as the entry point, common discovery tasks stay compact and easy to launch.
